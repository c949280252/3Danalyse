# 3D彩票分析功能说明

## 项目结构

单文件应用：`lottery_analysis.html`

五个标签页，**互不相干**，各自独立计算：

| 标签 | 玩法 | 必赢条件 |
|------|------|---------|
| 组选 | 组三/组六/豹子 | 三类各自盈利 |
| 复式 | 复式3-9码 | 各自盈利 |
| 跨度 | 跨度0-9 | 每个跨度单独盈利 |
| 和值 | 和值0-27 | 每个和值单独盈利 |
| 人工分析 | 自定义组合 | 根据买的组合计算 |

---

## 一、组选/复式/跨度/和值（原有功能）

### 1.1 组选标签

**三大类，每类只能填一项，三类都必须填**：

| 大类 | 包含玩法 | 数量 |
|------|---------|------|
| 组三 | 两码~九码/全包 + 沾边赖1~7码 | 16种 |
| 组六 | 四码~九码/全包 + 沾边赖1~7码 | 14种 |
| 豹子 | 全包 | 1种 |

**1000个号码的构成**：
```
1000个直码 = 组三(270个) + 组六(720个) + 豹子(10个)
```

**必赢条件**：三类各自盈利，无论开组三、组六还是豹子都不亏。
```
g3Win >= realCost && g6Win >= realCost && bWin >= realCost
```

### 1.2 跨度标签

10个玩法，全填才能计算。

**跨度计算**：最大数字 - 最小数字 = 跨度值

| 跨度 | 覆盖直码 |
|------|---------|
| 0 | 10 |
| 1 | 54 |
| 2 | 96 |
| ... | ... |
| 9 | 54 |

总计覆盖1000个号码。

### 1.3 和值标签

14个玩法（成对共用设置项），全填才能计算。

和值0-27共28个值，赔率对称的成对共用一个设置项。

## 二、人工分析标签（新增功能）

### 2.1 功能概述

允许用户自定义选择玩法、赔率和金额，分析是否必赢及庄家利润。

### 2.2 玩法分类

#### 需要输入号码的玩法

| 玩法 | 输入示例 | 说明 |
|------|---------|------|
| 组三N码 | "012" | N个不重复数字，生成6注组三 |
| 组六N码 | "0123" | N个不重复数字，生成C(N,3)×6注组六 |
| 沾边赖组六N码 | "13579" | 每个数字对应一组沾边赖，分开计算 |
| 复式N码 | "012" | N个数字，生成N³注 |

**切换玩法时**：号码框清空、赔率框清空、金额保留

#### 不需要输入号码的玩法（自动计算，号码框禁用显示"自动"）

| 玩法 | 覆盖直码数 |
|------|-----------|
| 豹子全包 | 10个 (000,111,...,999) |
| 组三全包 | 270个 |
| 组六全包 | 720个 |
| 跨度0-9 | 根据跨度值计算 |
| 和值0-27 | 根据和值计算 |

### 2.3 计算逻辑（枚举法）

#### 核心思想

枚举所有1000个直码(000-999)，计算每种情况的庄家利润，然后按概率加权平均。

#### 庄家利润公式
```
庄家利润 = 总投注 - 返水 - 赔款
```

- 不中：庄家收入全部投注，返水给玩家
- 中奖：庄家收入投注，返水，再赔给玩家

#### 庄家期望利润率
```
庄家利润率 = Σ(每种情况利润 × 概率) / 总投注 × 100%
```

#### 情况枚举示例

**单个玩法**：不中、中

**两个玩法（如同时买A组和B组）**：
- 不中、只中A、只中B、AB都中

**三个玩法（如同时买A组、B组、C组）**：
- 不中、只中A、只中B、只中C、AB都中、AC都中、BC都中、ABC都中

### 2.4 特殊情况处理

#### 沾边赖多组同时购买

例如同时买：
- A组六(13579)，赔率15，买100元
- B组六(24680)，赔率14，买100元

有4种情况：
- 不中：庄赢 200-200×9.7%
- 只中A：庄输 100×15-200+200×9.7%
- 只中B：庄输 100×14-200+200×9.7%
- AB都中：不可能（组六数字互斥）

按概率加权平均计算庄家利润率。

### 2.5 复式玩法

输入数字如"012"：
- 买012的任意排列（27注）
- 中奖条件：直码任意位在012中

## 三、使用说明

1. 打开 `lottery_analysis.html`
2. 选择标签页（组选/复式/跨度/和值/人工分析）
3. 填入赔率（人工分析还需填号码和金额）
4. 设置返水比例（默认9.7%）
5. 点击"分析"
6. 查看结果

### 组选输入规则
- 组三只能填一项（含沾边赖）
- 组六只能填一项（含沾边赖）
- 豹子必填
- 三类都必须填

### 跨度/和值输入规则
- 必须全填才能计算
- 预填了默认赔率，可直接分析
- 可修改个别值

### 人工分析输入规则
- 选择玩法（不同玩法填写规则不同）
- 根据提示输入号码/或自动
- 填写赔率和金额
- 可添加多行
- 点击"分析"查看各种情况的庄家利润
